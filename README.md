# Dual-Proximity-Sensor-Using-Arduino-Uno
This project uses an Arduino UNO with two HC-SR04 ultrasonic sensors — one on the left and one on the right — to measure distance on both sides independently. When an object moves further than 5cm away from either sensor, the matching LED on that side lights up, giving you a simple visual alert for each direction.
It's a great introduction to working with sensors, LEDs, and basic logic on the Arduino. Planing on Using this for cabinent lighting
